Structured Data Pipeline

Objective

The primary objective of this project is to gain practical skills in generating a data pipeline. The deliverables will involve creating a data pipeline with relational data using Azure's Database and Data Factory. I will be using the AdventureWorks data set published by Microsoft. Here's a README in notebook style that includes all CREATE tables and views queries.

Approach

  • My first step was to create a new schema that would contain the source and sink. I called the new schema "PipelineSchema"
  • I then created three views using the existing tables ib the existing schema.
  • After that, I had to make three tables that I can unload the data from the views in. The tables were modeled using the right data types and constraints as found in the views.
  • Lastly, I set up a new pipeline in Azure's Data Factory that copies data from the source (views) into the designated sink (tables).

Tech Stack

  • Microsoft Azure Database
  • Microsoft Azure Data Factory
